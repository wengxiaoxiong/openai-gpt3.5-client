## 使用方法
网站不需要部署，打开html可以直接使用的
这是一个个人版本的ChatGPT的客户端，使用之前需要自己去搞一个API-KEY，该怎么搞我就不多说啦，当然你可以打开网站直接使用网站自带的key
目前已经支持下载聊天记录，可以下载txt和markdown格式的。
